Buatkan issue.md untuk planning berikut:

Buat table users dengan kolom berikut:
- id INT autoincrement primary key
- email varchar(100) not null unique
- password varchar(255) not null (password merupakan hash dari bcrypt)
- created_at timestamp default current_timestamp
- updated_at timestamp default current_timestamp on update current_timestamp

Buatkan juga migration file untuk table users tersebut

kemudian buat API untuk regitrasi user baru:

}
